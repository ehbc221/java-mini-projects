Instructions :

1. Ouvir le projet dans Netbeans. (Si vous preferer Eclipse, il vous faudra créer le projet)

2.Assurez-vous d'avoir les librairies réquises dans votre classpath : SwingX, TopLink, MySQL Driver

3.Dans META-INF, éditez le fichier Persistance.xml afin qu'il s'accorde avec vos configurations de MYSQL

4.Lancez l'application
